# AI Multi-Agent Code Review System

Automated code review using specialized AI agents for security, performance, and quality analysis.

## Features
- Parallel multi-agent review (security, performance, quality)
- Git diff analysis
- Structured markdown reports
- Zero-config local execution

## Usage
\`\`\`bash
pip install -r requirements.txt
python main.py /path/to/repo HEAD~1 HEAD
\`\`\`

## Configuration
Update `config.py` with your SIM.ai agent endpoints.
